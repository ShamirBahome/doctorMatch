# 🏥 Guelph Doctor Finder

A web app that helps Guelph, Ontario residents find family doctors who are accepting new patients.

The Problem

Over 30,000 people in Guelph don't have a family doctor. There is no easy way to find one.

✅ The Solution

A free platform where residents can:
- Search for doctors by name, language, or gender
- See which doctors are accepting new patients
- Read and write community reviews
- Find walk-in clinics with wait times

 Built With

- **Python** (Flask)
- **SQLite** (Database)
- **HTML + Tailwind CSS** (Frontend)
